SHERF: Generalizable Human NeRF from a Single Image

介绍

目标：旨在从 2D 观察中恢复高质量的 3D 人体，避免捕获真实 3D 几何信息的需要

怎么做：我们提出了一个分层特征库来解决单个图像输入中信息丢失的挑战。分层特征库捕获了全局人体结构和局部精细细节，引入了特征融合变压器来有效地合并分层特征库中的特征，我们的方法可以为可见区域重建正确的颜色，并为不可观察区域提供合理的猜测。

为了解决可动画性的挑战，SHERF 在规范空间中对 3D 人体表示进行建模，使其能够进行姿势变换和渲染。

我们使用SMPL先验[47]将从输入图像中提取的层次特征转换到规范空间，在规范空间中对它们进行编码以更好地补全缺失信息并获取人体结构信息

数据集： 包括 THuman [83]、RenderPeople [1]、ZJU_MoCap [60] 和 HuMMan [9]。

在这项工作中，我们应用 SMPL(pose θt and shape βt) 的线性混合蒙皮 (LBS) 算法将点从规范空间变换到目标/观察空间

Gk(θ, J)是k的变换矩阵，wk是混合权重

全局特征fglobal、点级特征fpoint和像素对齐特征fpixel

fglobal：compress the input image into a compact latent code using a 2D encoder，捕获全局结构和整体外观

fpoint:先通过将 SMPL 端点投影到输入图像的 2D 特征图来提取每个点的特征，从输入中我们只能看到一半的端点，为了使点级特征意识到遮挡，我们只提取从当前相机 Po 可见的顶点的特征，然后，我们执行逆 LBS 将姿势顶点特征变换到规范空间，然后将其体素化为稀疏 3D 体积张量，并通过稀疏 3D 卷积进一步处理，从编码的稀疏 3D 体积张量中，我们可以提取点 xc 的点级特征 fpoint(xc)，人体结构和局部纹理细节

fpixel：然而，由于 SMPL 网格分辨率和体素分辨率有限，它可能会遭受严重的信息丢失。为了补偿细粒度局部信息缺失问题，我们通过将 3D 变形点 xc 投影到输入视图中来进一步提取像素对齐特征

我们通过LBS将观察空间中的变形点xc变换为xo = LBS(xc; θo, βo)并将其投影到输入视图，以便可以查询像素对齐的特征fpixel(xc)，可以表示为

W是2D特征编码器，Π(·)表示3D到2D投影算子

为了避免过度拟合无信息的像素对齐特征，我们根据相应的 3D 变形点 xc 与其最近的 SMPL 顶点之间的距离为像素对齐特征分配不同的权重*。

Feature Fusion Transformer

对于可观察的部分，我们应该更多地依赖像素对齐的特征来实现最好的纹理恢复水平。而对于不可见部分，全局特征和点级特征（其中编码了更连贯的 3D 感知信息）应该贡献更多。为了建模这种复杂的特征关系，我们采用自注意力模块 Attn(·)，具有三个注意力头以进行有效的特征融合


融合后的特征被输入到 NeRF 解码器中，以预测点 xt 的密度 σ 和 RGB c

SHERF包含五个可训练模块，分别是三个层次特征提取模块、特征融合变压器和NeRF解码器，使用四个损失函数来监督训练

实验

数据集：四个大型人类数据集进行评估，即 THuman 、RenderPeople 、ZJU_MoCap 和 HuMMan 。

我们的 2D 编码器采用预训练的 ResNet18 主干来提取 1D 向量 f ∈ R512 用于全局特征提取，以及特征图 f ∈ R64×256×256 用于点级和像素对齐特征提取，为了保留更多低级信息，我们进一步对 RGB 值进行位置编码，将它们附加到 2D 特征图以形成特征图 f ∈ R96×256×256

The Mapping Network and Style-Based Encoder 采用 EG3D [11].

四层稀疏卷积[16]用于提取96维点级特征。

查询的全局特征、点级特征和像素对齐特征被连接并投影到 32 个通道，然后输入特征融合转换器

评估指标

定量评估渲染的新颖视图和新颖姿势图像的质量：PSNR SSIM LPIPS

定性：

消融：验证所提出的分层特征提取组件和特征融合转换器的有效性，我们随后添加不同的组件并评估 THuman 数据集上的性能
