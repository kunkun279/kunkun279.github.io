计算子串出现次数
发表于2022-06-06|ACM
﻿ 题目描述给定完全由小写字母组成的两个字符串a和b，求b在a中出现的次数 输入第一行为字符串a； 第二行为字符串b; 输出一个整数，表示b在a中出现的次数 样例输入 sharonisagoodgirlihopeshecangetgoodgradesinthefinalexamsin 样例输出 2 c++中，我常用substr(n,m)函数(从n开始截取长为m的部分字符串)来截取字符串长度，来比对截取部分的代码与子串是否相同。代码：#include<iostream>#include<string>using namespace std;int main(){ string a; string b; int count=0; cin>>a; cin>>b; for(int i=0;i<a.length();i++) { if(a.substr(i,b.length())==b) { count++; } } cout<<count<<endl; return 0;}
